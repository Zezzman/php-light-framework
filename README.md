# PHP Light Framework

Basic framework that allows you to route, launch, render and authorisation through accessing it via namespace System, api requests and cli access.

Basic framework features:
* Routing
* Launch
* Render View
* Authorisation
* Static Views
* Database Access

Access Points:
* Namespace System
* API requests
* CLI requests.
